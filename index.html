Proyectos

JUEGO DEL AHORCADO

Segundo challenge de Alura correspondiente al programa Oracle Next Education. El desafío fue crear una aplicación que permita jugar al ahorcado y cumpla los siguientes requisitos:

  • Admita sólo letras mayúsculas, sin caracteres especiales
  • Tener una función "agregar palabra"
  • Tener opción de abandonar juego o "desistir"
  • Posibilidad de ver las letras elegidas en pantalla, las cuales no podían repetirse
  • Mostrar resultado del juego una vez finalizado

El proyecto fue desarrollado a partir de los conocimientos adquiridos en la formación Principiante en Programación y se llevó a cabo mediante la utilización de la aplicación "Trello".

ENCRIPTADOR DE TEXTO

Primer challenge de Alura correspondiente al programa Oracle Next Education. El desafío fue crear una aplicación que permita encriptar y desencriptar un mensaje de texto, además de cumplir los siguientes requisitos:

  • Admitir sólo letras minúsculas, sin caracteres especiales
  • Tener una función borrar entrada y salida
  • Tener función de desencriptar
  • Tener un botón para copiar salida de mensaje

El proyecto fue realizado luego de la formación en Lógica de Programación realizada en la plataforma de Alura Latam.
